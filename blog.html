The localStorage and sessionStorage has the same purpose with difference mechanism. The differences between localStorage and sessionStorage is localStorage can only be clear when the user manually clean it or the application clean it. But the sessionStorage is cleaned when the browser or tab closes.

Any variable declared outside function and curly brackets have Global Scope. Global Scope means that the variable can be accessed anywhere from the program. On the other hand Block scope is the space where any varible is inside curly brackets. The varibale declared inside the Block Scope can only be accessed from the Block Scope.

Event Loop is a javascript runtime model. It has two simple job. One is monitor call stack and another is callback queue. When the call stack is empty, the event loop will push first event in call stack from the queue and which will execute. This iteration process is called as tick in event loop.

A vairable which is not assigned to any value is undefined. Function returns undefined if a value was not returned. Method or statement also returns undefined if any variable which is being evaluated doesnot has any assigned value.
